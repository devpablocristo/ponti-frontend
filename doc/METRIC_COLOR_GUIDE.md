# Metric Color Guide

Paleta oficial para métricas (barra superior y subtexto) en `IndicatorCard`.

## Grupos y colores

- Superficie / Hectáreas -> `amber` `#b45309`
- Costos / Dinero (USD) -> `red` `#dc2626`
- Rendimiento productivo -> `green` `#166534`
- Consumos físicos / Inventario -> `gray` `#9ca3af`
- Progreso / Estado (%) -> `blue` `#1e3a8a`
- Financiamiento / Aportes -> `purple` `#5b21b6`

## Regla de uso

- Usar siempre el color por **tema** (no por pantalla).
- El valor principal (número) se mantiene en negro para legibilidad.
- La barra superior y el subtexto usan el color temático.

## Referencia técnica

- Componente: `ui/src/components/Card/IndicatorCard.tsx`
- Mapa de colores: `barColors`
